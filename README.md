HASAD is a smart farming system designed to support small farmers through virtual sensing, data-driven services. The complete HASAD system is planned to provide the following three core services:

1. Irrigation Recommendation emgine:
    Provides reccomendations on optimal irrigation timing and quantity based on real time weather data.
2. Disease Risk Prediction Service:
    Predicts crop disease risks early by an ML model with the data collected from the weather.
3. Agricultural AI Assistant:
    Offers interactive advisory support by answering farmer questions and providing agricultural guidance.
